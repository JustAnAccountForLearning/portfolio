Twitter Map

This started as a final project for UVM's Software Engineering course. I worked with a team of three to develop a visualization for Tweets throughout the USA. Tweets are sourced through a 1% hose and formatted using Bingâ€™s geocoding API. Tags can be selected by the user from a list created from the database tags.
